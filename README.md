# Event Horizon

# Purpose

A website dedicated to showing specific events, concerts, and shows in a select radius near you and also a general weather app based on longitude and latitude with a five-day forecast and local storage.

## Built With

- HTML
- CSS
- Tailwind CSS
- Javascript
- openweatherAPI
- seatgeekAPI
- axios cdn
